Smart.pr Labs

Changelog

A simple list of what we built in Labs. Everything here was made in October 2025 to try ideas and see what actually helps.

Version 0.3.1 October 2025

Goal: Help teams land stronger open rates without leaving the composer.

  • Subject Line Coach adds a “Generate subject” button beside the subject field, using the live mailing content to draft a PR-friendly line.
  • Polished the overview copy so Angle Assistant focuses on story ideation.
Version 0.3.0 October 2025

Goal: Let teams try small ideas quickly without waiting for a full release.

  • Paragraph Writer suggests a short follow-up paragraph in the Bee editor to keep drafts moving.
  • PR Feedback Coach gives quick checks on clarity and tone before sending.
  • Angle Assistant helps find alternative story angles and spin up press releases faster.

Experiments are on by default. You can switch them off from the overview.

Version 0.2.0 October 2025

Goal: Make AI usage in Labs visible so you know what’s being used and roughly what it costs.

  • Added a small usage dashboard with request counts, token totals, and cost estimates.
  • Feature toggles and API keys now sync across browsers and profiles.
Version 0.1.0 October 2025

Goal: Set up one place to try experiments and collect early feedback.

  • Released the Labs overview with quick toggles and API key setup.
  • Opened the first beta of Angle Assistant for early testers.
